# Skill: create-blog-content

## When to Use
Use this skill when:
- Writing a new article for `src/content/posts/`.
- Enhancing old blog posts or updating metadata descriptions.
- Designing content templates.

## Principles
1. **Value First**: Write in a direct, conversational, human-expert tone (no AI clichés).
2. **Metadata Correctness**: Complete all required frontmatter elements (title, date, author, category, description, image).
3. **Internal Linking**: Interlink to relevant service pages and portfolio projects.

## Task Checklist
- [ ] Write clear frontmatter header.
- [ ] Construct semantic headings (`##`, `###`) and highlight key code/workflow snippets.
- [ ] Connect a clear Call to Action at the end of the post.
