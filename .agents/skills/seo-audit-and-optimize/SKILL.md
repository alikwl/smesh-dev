# Skill: seo-audit-and-optimize

## When to Use
Use this skill when:
- Creating a new service description, blog post, or resource page.
- Modifying layouts, headers, or metadata handlers.
- Auditing site structure for search engine indexability.

## Principles
1. **Generative Engine Friendliness**: Every page needs a structured summary at the top to assist AI engines.
2. **Author & Organization Entities**: Embed schema referencing the founder (Haider Ali) and parent agency (BismillahTC Digital Services) to build E-E-A-T trust factors.
3. **Structured Schemas**: Integrate JSON-LD schemas directly onto the page dynamically based on content.

## Task Checklist
- [ ] Verify standard schema output using `<Schema />` and `<SEO />` components.
- [ ] Check headings logic (only one `<h1>` per page, clean descending tree).
- [ ] Ensure valid title and meta description attributes exist.
