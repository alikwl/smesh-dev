# Skill: conversion-optimization

## When to Use
Use this skill when:
- Designing or modifying lead capture forms, pricing components, or CTA sections.
- Modifying resource locking workflows (`ResourceGate.astro`).
- Creating context-matching services recommendation layouts.

## Principles
1. **Contextual Interlinking**: Automatically present corresponding service CTAs relative to the page content category.
2. **Low Friction**: Keep form fields minimal. Standard fields: Name, Phone (WhatsApp), Email, and Project Description.
3. **Persuasive Proofing**: Display customer testimonials and estimated values prominently to establish credibility.

## Task Checklist
- [ ] Ensure CTAs are readable, distinct, and visually highlighted using accent colors.
- [ ] Confirm form validation feedback displays clear error/success states.
- [ ] Check that modal triggers execute client-side without layout flashes.
